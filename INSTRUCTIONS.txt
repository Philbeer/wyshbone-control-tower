================================================================================
WYSHBONE STATUS DASHBOARD - SETUP INSTRUCTIONS
================================================================================

Thank you for using the Wyshbone Status Dashboard! Follow these simple steps
to get started monitoring your Wyshbone applications.

================================================================================
STEP 1: UPDATE CONFIGURATION
================================================================================

Open the file: config/sources.json

Replace the placeholder values with your actual Wyshbone app information:

  - "baseUrl": Replace with your Wyshbone app's full URL
    Example: "https://wyshbone-ui-yourname.repl.co"
    
  - "exportKey": Replace with your app's export authentication key
    Example: "sk_abc123xyz456..."

You can add more sources by adding additional objects to the JSON array.

EXAMPLE:
--------
[
  {
    "name": "My Wyshbone UI",
    "baseUrl": "https://wyshbone-ui-john.repl.co",
    "exportKey": "sk_live_abc123"
  },
  {
    "name": "My Supervisor",
    "baseUrl": "https://wyshbone-supervisor-john.repl.co",
    "exportKey": "sk_live_xyz789"
  }
]

================================================================================
STEP 2: START THE SERVER
================================================================================

Run this command in your terminal:

    node server.js

The server will:
  âœ“ Load your configuration
  âœ“ Start polling your Wyshbone apps every 2 minutes
  âœ“ Display the dashboard URL

================================================================================
STEP 3: ACCESS THE DASHBOARD
================================================================================

Once started, you'll see output like:

    âœ“ Server running on http://localhost:3000

Open your browser to:

    http://localhost:3000/status

The dashboard will show:
  - Status of each configured Wyshbone app
  - Current metrics (cleverness, LOC, TODO/FIXME counts)
  - Changes since the last check (deltas)
  - Recent activity across all apps
  - Auto-refreshes every 60 seconds
